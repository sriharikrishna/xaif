XAIF 1.0 Schemas Documentation

  • Top-level schema: this defines the input to xaifBooster; definitions are in  xaif.xsd and xaif_base.xsd
  • Inlinable intrinsics schema: this describes (language specific) inlinable intrinsics and their partial derivatives read by xaifBooster;  definitions are in  xaif_inlinable_intrinsics.xsd
  • Transformation output schema: this is the basic XAIF schema extended by derivative propagators (see in particular  xaif_derivative_propagator.xsd)  and elements for inlinable subroutines as well as code replacement in templates; definitions are in xaif_output.xsd
  • Noninlinable intrinsics schema: this is an as of now incomplete schema to handle black-box routines; definitions are in  xaif_noninlinable_intrinsics.xsd
